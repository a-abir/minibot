Setup
======
*In order to start coding download the package minilib*

Install git
***********

*First lets install git*

``sudo apt install git``

Download package
****************

*Next lets downlad the package*

``git clone https://github.com/atikul99/minibot``

Install package
****************

*Next lets install the package*

``cd minibot``

``pip3 install --user .``


Run all at once
***************

.. code-block:: bash

   sudo apt install git
   git clone https://github.com/atikul99/minibot
   cd minibot
   pip3 install --user .
   # miniLib installed
